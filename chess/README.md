# Chess Game

This is a simple implementation of a chess game in Python using Pygame. It's a work in progress and not yet fully complete.

## Current Features

- Basic chess board and piece setup
- Piece movement according to chess rules
- Pawn promotion
- Castling

## Incomplete Features

The following features are not yet implemented:

- Check and Checkmate detection: The game currently does not detect check or checkmate situations.
- Draw conditions: The game does not recognize draw conditions such as stalemate, threefold repetition, or the fifty-move rule.
- AI opponent: Currently, the game only supports two human players. An AI opponent is not yet implemented.

To run the game, execute the `main.py` script:
